# CURA Healthcare Service

## Project Overview
CURA Healthcare Service is a healthcare appointment management application that enables users to securely access the system and schedule medical appointments at selected healthcare facilities. The project focuses on reliability, usability, and correct handling of user inputs.

## Objectives
- Validate user access and system behavior
- Ensure smooth appointment scheduling flow
- Verify input validations and error handling
- Ensure overall system stability and quality

## Key Features
- User authentication
- Appointment scheduling with facility and date selection
- Support for healthcare programs
- Input validation and confirmation messages
- Error handling and feedback mechanisms
